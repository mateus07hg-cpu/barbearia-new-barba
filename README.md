# 💈 Barbearia New Barba

## 📌 Descrição
Este projeto foi desenvolvido para uma atividade acadêmica do curso de Análise e Desenvolvimento de Sistemas.

O sistema simula uma tela de seleção de serviços de uma barbearia, onde o usuário pode adicionar serviços e visualizar o valor total em tempo real.

---

## 🚀 Funcionalidades
- Adicionar serviços (Corte, Barba, Sobrancelha)
- Atualização automática do total
- Lista de pedidos dinâmica
- Finalização de pedido com limpeza da lista

---

## 🛠️ Tecnologias Utilizadas
- HTML
- CSS
- JavaScript

---

## 📚 Conceitos Aplicados
- Manipulação do DOM (querySelector, getElementById)
- Eventos (addEventListener)
- Arrow Functions
- Conversão de dados (parseFloat)
- Estruturação com Grid no CSS

---

## ▶️ Como Executar
1. Baixe ou clone o repositório
2. Abra o arquivo `index.html` no navegador
3. Clique nos botões "Adicionar"
4. Veja o total sendo atualizado em tempo real
